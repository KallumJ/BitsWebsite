Simple Voice Chat

We make use of a mod called Simple Voice Chat, which adds proximity voice chat to Minecraft, so you can talk to other nearby players! It's mod page can be found here to get yourself a download.

If you have never installed mods before, or need a refresher, here are some instructions, just select the launcher of your choice.

Installation Instructions:

  1. Head to fabricmc.net and download the latest fabric installer and run it.
  2. Ensuring the Minecraft Launcher is closed, and that the selected Minecraft version (underlined in red) is correct, hit install.
  3. If all goes well, a success message like this will be shown. Dismiss it, close the installer, and open the Minecraft Launcher.
  4. Under the "Installations" tab in the Minecraft Launcher (underlined in blue), you should see a new install called something like fabric-loader-MC.VERSION.NUMBER. This name is a bit ugly, so hit the button with 3 dots, then edit, and give it a nicer name, and hit save.
  5. Again in the "Installations" tab, click the open folder button (boxed in blue)
  6. If it doesn't exist already, create a folder in the folder that got opened called mods
  7. Drag your download for Simple Voice Chat into the mods folder. NOTE IF YOU ALREADY HAD A MODS FOLDER: if you already had a mods folder, you may already have other mods installed. These mods will need to be Fabric mods, and work with whatever version your trying to play. If they are incompatible, remove them.
  8. Ensuring that the installation we just set up is selected in the installation selector (highlighted in blue), hit play.
  9. Finally, dismiss this safety message. It's nothing to worry about. It's just Mojang warning they can't gurantee safety features like player muting will work (they will :D). Thats all! Now Simple Voice Chat is installed!
  10. You can verify you installed everything properly by loading up a world and pressing the V key. It should bring up this settings menu for Simple Voice Chat, confirming everything worked!

  2. In the window that pops up, press "Install Fabric" (highlighted in blue).
  3. In the window that pops up, ensuring the starred version is selected, hit "Ok".
  4. Click "Open .minecraft" (highlighted in blue)
  5. Drag your download for Simple Voice Chat into the mods folder.

If you have played modded Minecraft recently, chances are you have used Curseforge to download your modpacks. Curseforge can also be used to manage "vanilla" installations, so here are some instructions if you want more of a "1-click-experience."

  1. Click "Create Custom Profile" (highlighted in blue).
  2. Give your profile a nice name, ensure the Minecraft version (highlighted in red) is correct, and change the "Game Type" toggle to "Fabric" (highlighted in blue), and press create.
  3. Right click the profile you created, and select "Add More Content" (highlighted in blue).
  4. Search Simple Voice Chat in the search bar, and hit install on the mod by henkelmax, highlighted in blue below.
  5. Thats it! Launch your profile and you'll now have a Minecraft instance with Simple Voice Chat installed!
  6. You can verify you installed everything properly by loading up a world and pressing the V key. It should bring up this settings menu for Simple Voice Chat, confirming everything worked!

Custom Plugin

Our custom plugin ensures allows us to add features we think are right for the server. It's highlight features are listed below:

  • /beam [player] - Requests to teleport to the provided player
  • /bed - Teleports the player back to the bed they last slept in
  • /rtp - Teleports the player to a random location on the map
  • /warp [warp] - Teleports the player to the specified warp. Only used for community areas, such as Spawn, Nether Hub, etc.
  • /nick [nickname] - Gives you the specified nickname
